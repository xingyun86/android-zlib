CodeBlocks安装目录：D:\DevelopmentEnvironment\CodeBlocks
Android NDK-BUNDLE目录：D:\DevelopmentEnvironment\CodeBlocks\ndk-bundle
解压CodeBlocks编译器配置选项AppData-CodeBlocks(AndroidNDKBuild).rar中的CodeBlocks目录解压到目录：%APPDATA%
修改CodeBlocks编译选项方法：
Setting->Compiler..->Compiler Setting->Other settings->Advanced options..->Edit advanced compiler settings->Yes->Advanced compiler settings->Commands->Link object files to dynamic library->Command line macro
修改命令行中的选项即可
